## altcorp.org

## installation

```
git clone https://github.com/kenjitheman/altcorp.org
```

- install all dependencies

```
yarn install
```

## usage

- run

```
yarn start
```

- run in dev mode

```
yarn dev
```

## contributing

- pull requests are welcome, for major changes, please open an issue first to
  discuss what you would like to change
